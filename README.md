## ✅ Running the app
```
server/config/dev.js 파일 생성
server/config/s3.json 파일 생성
server/config/iamport.json 파일 생성
root directory에 uploads/thumbnails 폴더 생성
root directory에 uploads/watermark 폴더 생성

$ npm install
$ cd client
$ npm install
$ cd ..
$ npm run dev
```
